# PrayerOffline

Privacy-first offline Muslim Prayer Time app built with Flet.

## Features
- Accurate prayer times
- Jamaat adjustments
- Real-time dashboard
- Local notifications
- Tasbih counter
- Multilingual support
- Offline-first, privacy-focused

## Setup
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Run
```bash
python main.py
```
